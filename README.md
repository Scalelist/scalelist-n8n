# n8n-nodes-scalelist

This is an n8n community node to call Scalelist API services.

[n8n](https://n8n.io/) is a fair-code licensed workflow automation platform.

## Installation

Follow the [installation guide](https://docs.n8n.io/integrations/community-nodes/installation/) in the n8n community nodes documentation.

## Operations

*   **Scalelist API**: Access various endpoints of the Scalelist API.

## Credentials

You will need an API key from Scalelist to use this node.

## Compatibility

*   n8n: 0.111.0 or later
